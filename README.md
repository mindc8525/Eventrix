# Eventrix

Eventrix is a campus-wide event management platform that brings together all clubs, councils, cells, and fests into one unified space. Students can browse entities, view upcoming events, subscribe for reminders, and manage their favourites. Admins and coordinators can create, update, and delete events with ease.
